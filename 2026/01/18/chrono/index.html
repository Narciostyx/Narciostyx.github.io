C++11 chrono 时间库的时钟、时间点与时间段

  • 跟ctime类似，是表示与处理时间的标准库，包含三大功能时钟、时间段与时间点
  • 时钟

Note
对于上述时钟类型，其内部均含有一静态方法now获取该时钟对应的时间点，当前时间

  • 对与系统时钟还有两个静态方法，to_time_t与from_time_t，即转换成ctime的time_t类型或从其转换为time_point
  • 时间点，即template<class Clock,class Duration = typename Clock::duration> class time_point

Note
使用时间点可以计算经过的时间，即获取时间段duration

  • 时间段，即template<class Rep,class Period=std::ratio<1>> class duration，前一模板参数代表刻度计数，即允许该模板实例化的参数，如std::chrono::duration<double> half_second(0.5)，即0.5乘以后一模板参数刻度周期（默认为1，代表1s）为0.5秒，因此std::chrono::duration<int,std::ratio<60>> minute(1)代表一分钟
  • 对于duration时间段对象进行的四则运算或比较运算，均会返回一个新的duration对象
