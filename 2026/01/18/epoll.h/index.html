epoll函数

Linux epoll IO 多路复用事件驱动机制详解

  1. 提供IO多路复用技术的一个函数epoll，该函数使用事件驱动，主要使用三个函数epoll_create，epoll_ctl，epoll_wait
  2. EPOLL_EVENTS，即epoll事件枚举类型，如：
    1. EPOLLIN，读事件
    2. EPOLLOUT，写事件
    3. EPOLLRDHUP，套接字关闭或半关闭
    4. EPOLLPRI，紧急数据可读事件
  3. epoll_data，联合体类型，声明如下：
  1. epoll_event，保存当前监听的事件epoll_events以及其指向的epoll_data用户数据
  2. int epoll_create(int size)，创建一个epoll对象并返回其fd，需要使用close关闭该描述符，size参数一般被忽略（仅要求其值大于0即可）
    • 本质是创建一个eventpoll对象并初始化，大致为红黑树与双向链表、互斥锁的组合
  3. int epoll_ctl (int __epfd, int __op, int __fd, struct epoll_event *__event) __THROW，控制epoll实例，按参数顺序为：
    1. epoll对象fd
    2. epoll操作，也即红黑树上的操作，即 EPOLL_CTL_ADD，EPOLL_CTL_DEL，EPOLL_CTL_MOD
    3. 监听的socket的fd
    4. 事件，这里指添加进的事件信息，一般为struct epoll_event
  4. int epoll_wait (int __epfd, struct epoll_event *__events,int __maxevents, int __timeout)，等待在epoll实例上发生的事件，__time指定等待事件，单位为毫秒，若为-1则一直阻塞，这里的事件一般是一个事件数组即struct epoll_event []
