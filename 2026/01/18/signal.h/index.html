signal函数及其信号集

C语言信号处理库 signal.h 的信号类型与常用函数

  1. 定义类型sig_atomic_t与sigset_t，前者为原子实体，后者表示信号集
  2. 信号类型：

  3. 常用的库函数：
  • void (*signal(int sig, void (*func)(int)))(int)，设置信号处理程序，该函数接受一个信号sig，以及void (*)(int)信号函数指针func，该函数本身返回一个信号函数指针
  • int raise(int sig)，向当前进程发送信号
  • int kill(pid_t pid, int sig)，向指定进程发送信号
  • int pause(void)，挂起进程直到捕捉到信号

Note
一般来讲，Linux下对大多数信号的默认处理程序是terminate()
